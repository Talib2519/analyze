About This Project

Powerful Data Processing

Our `execute.py` script, powered by Pandas, efficiently processes structured data from `data.xlsx` (converted to `data.csv`). It's designed to handle common data anomalies, ensuring accurate and reliable results. A non-trivial error fix ensures robust handling of mixed data types and missing values in critical columns.

Automated CI/CD with GitHub Actions

Every push to the repository triggers a GitHub Actions workflow. This pipeline lints the code with `ruff`, runs the `execute.py` script, and then publishes the generated `result.json` directly to GitHub Pages. This ensures continuous quality and up-to-date insights.
